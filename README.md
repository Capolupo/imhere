# imhere

Neste projeto estudei a diferença entre ScrollView e FLATLIST, estou usando o FlatList.
Normalmente o ScrollView é utilizado para uma lista pequena, pois o react sempre renderizaa lista inteira, 
já o FlatList é renderizado conforme a necessidade durante a rolagem.

Neste projeto tambem utilizo Alerts com opções de escolha ( SIM / NAO ) e utilizo estados.
